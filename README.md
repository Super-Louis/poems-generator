# poems-generator
__generate poems with lstm__
- train model  
_execute `python3.6 train.py` in the terminal to train the lstm model._
- compose poems  
_run the `generate_poems.py` script in the terminal to compose poems as you like._
- sample  
秋风度月寒，风雨日新恩。  
野泪双云手，碑麻亦立身。  
霏和浑落即，去火侔南汐。  
不衣金虎首，艺身洁稷耕。
